Flexbox Column

Use this create a Flexbox column layout.

        .column{
          display: flex;
          flex-direction: column
        }
CSS Grid Layout

Build a 12-column layout using CSS Grid

        .grid{
          display: grid;
          width:100%;
          grid-template-colums:
          repeat(12, 1fr);
        }
